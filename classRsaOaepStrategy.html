Detailed Description

RSA-2048-OAEP asymmetric encryption strategy (L3 concrete).

Used exclusively for AES key wrapping during key exchange — not for bulk data encryption. Inherits from IAsymmetricStrategy (L2).

encrypt(data, publicKeyPem) → delegates to encryptWithPublic() decrypt(data, privateKeyPem) → delegates to decryptWithPrivate()
