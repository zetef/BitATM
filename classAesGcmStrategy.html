Detailed Description

AES-256-GCM symmetric encryption strategy (L3 concrete).

Implements ISymmetricStrategy using OpenSSL EVP.

Wire format produced by encrypt(): [12-byte IV] [ciphertext] [16-byte GCM auth tag]

decrypt() verifies the auth tag — throws CryptoException if tampered.
