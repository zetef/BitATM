Detailed Description

template<typename T>
class IRepository< T >

Abstract CRUD interface for a persistent store of domain objects.

Template Parameters
TDomain class that maps to a database table (must inherit IEntity).

Concrete repositories (UserRepository, MessageRepository, etc.) inherit this and implement each method via Poco::Data::Session queries.
