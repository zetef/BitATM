Detailed Description

Handles KEY_EXCHANGE packets. L3 under BaseAuthHandler.

Two sub-operations in one packet type:

  • Upload own public key: packet.key contains the PEM public key.
  • Request peer's public key: packet.to is set, server replies with a KEY_EXCHANGE packet carrying the peer's public key in packet.key.
